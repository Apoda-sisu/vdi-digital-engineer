## 设计范围

1. **消防用水量**：室外消火栓 + 室内消火栓 + 自动喷水 + 其他灭火系统
2. **消火栓系统**：系统形式（临时高压/稳高压/低压），环状管网，消火栓布置
3. **自动喷水灭火系统**：系统选型（湿式/干式/预作用），喷头布置，报警阀组
4. **消防水源**：消防水池（容量 + 补水）、高位水箱、消防水泵房
5. **消防泵选型**：流量/扬程/功率，备用泵配置

## 公式调用协议

> **所有工程计算必须通过 `vdi_calculate` MCP 工具执行，禁止脑算。**

### 常用公式速查

| 计算场景 | 公式 ID | 说明 |
|----------|---------|------|
| 消防水池容积 | WA-FIR-001 | V = (Qf-Qb)×T |
| 消火栓设计流量 | WA-FIR-002 | Qx = n × qx |
| 水泵轴功率 | WA-EQ-001 | P = ρgQH/η |
| 水泵电机功率 | WA-EQ-002 | Pm = P/ηm |
| 管道水头损失 | WA-HYD-001 | 海曾-威廉姆斯公式 |

### 调用流程

1. 根据计算场景查找上表公式 ID
2. 调用 `vdi_search_formulas(query="<场景>", discipline="water")` 确认公式
3. 调用 `vdi_calculate(formula_id="<ID>", inputs={...})` 执行
4. 将返回的 `audit.evidence_tag` 写入设计输出引用

## 设计要点

| 项目 | 参数 |
|------|------|
| 室外消火栓流量 | 按建筑类别/体积查 GB 50974 表 3.3.2 |
| 室内消火栓流量 | 按建筑高度/类别查 GB 50974 表 3.5.2 |

| 高位水箱 | ≥ 初期火灾 10min 用水量 |
| 消火栓间距 | ≤ 60m（室外），≤ 30m（室内） |
| 自喷危险等级 | 按 GB 50084 附录 A 确定 |
