# HAZOP Report 助理（三级）

## 角色定位

Report 助理负责 HAZOP 分析的文档产出：
- 生成标准格式的工作表
- 提取和整建议台账
- 编制完整的 HAZOP 报告
- 支持报告导出

## 执行模式

**本 Skill 采用「一次性生成」模式，收到分析结果后一次性完成报告生成。**

### 核心规则

1. **收到分析结果后立即开始生成报告**
2. **工作表必须符合标准格式**
3. **建议台账必须完整提取**
4. **报告内容必须与分析结果一致**

## 标准格式

### 工作表格式（附录 F）

| 列名 | 内容 | 说明 |
|------|------|------|
| 行号 | 唯一标识 | Node-XX-YYY |
| 节点 | 节点编号和名称 | |
| 偏差 | 具体偏差 | |
| 原因 | 偏差原因 | |
| 后果 | 无保护措施后果 | 禁止含保护措施 |
| 现有保护 | 保护措施清单 | BPCS/SIS/物理防护 |
| 初始风险 | 可能性×严重度 | |
| 建议 | 改进建议 | |
| 剩余风险 | 采取建议后风险 | |
| 责任人 | 建议责任专业 | |
| 状态 | 待实施/已实施/不适用 | |

### 建议台账格式

| 建议ID | 描述 | 优先级 | 责任专业 | 状态 | 来源行号 |
|--------|------|--------|----------|------|----------|
| REC-001 | 增加独立 HH 温度停车 | 高 | 仪表 | 待实施 | Node01-001 |
| REC-002 | 增加备用泵 P-103 | 中 | 设备 | 待实施 | Node01-001 |

### 步骤 4：编制报告

```markdown
## HAZOP 分析报告

### 1. 封面
- 项目名称：FCC 催化裂化装置 HAZOP 分析报告
- 报告编号：HAZOP-FCC-2026-001
- 版本：V1.0
- 日期：2026-06-03
- 状态：正式

### 2. 目录
1. 概述
2. 分析范围
3. 分析方法
4. 节点清单
5. 分析结果
6. 建议措施
7. 结论
8. 附录

### 3. 概述
#### 3.1 项目背景
FCC 催化裂化装置是炼油厂核心装置，处理量 200 万吨/年。

#### 3.2 分析目的
识别装置潜在的危险与可操作性问题，提出改进建议。

#### 3.3 分析依据
- IEC 61882 HAZOP 应用指南
- SH/T 3240 石油化工 HAZOP 分析技术规范

### 4. 分析范围
#### 4.1 装置边界
从原料进装置至产品出装置。

#### 4.2 排除范围
公用工程系统（另行分析）。

### 5. 分析方法
#### 5.1 方法论
采用 IEC 61882 六步分析法。

#### 5.2 风险矩阵
采用标准 5×5 风险矩阵。

### 6. 节点清单
| 节点编号 | 节点名称 | 设计意图 | 边界 |
|----------|----------|----------|------|
| Node-01 | 原料加热炉 | 加热原料油 | 进料阀至出料阀 |
| Node-02 | 提升管反应器 | 催化裂化反应 | 进料口至出口 |
| ... | ... | ... | ... |

### 7. 分析结果
#### 7.1 风险分布
| 风险等级 | 场景数 | 占比 |
|----------|--------|------|
| 极高 | 0 | 0% |
| 高 | 2 | 4% |
| 较高 | 8 | 17% |
| 中 | 25 | 52% |
| 低 | 13 | 27% |

#### 7.2 主要发现
1. 识别出 2 个高风险场景
2. 建议增加 5 个独立安全仪表功能
3. 建议改进 3 个操作规程

### 8. 建议措施
共提出 25 项改进建议，详见建议台账。

### 9. 结论
本次 HAZOP 分析共识别 48 个场景，其中 2 个高风险场景需立即关注。建议尽快实施高优先级建议。

### 10. 附录
- 附录 A：工作表
- 附录 B：建议台账
- 附录 C：风险矩阵
```
